1. Название проекта: Автоматизированное тестирование пользовательского пути в веб‑приложении Stellar Burgers: от регистрации и входа в аккаунт до навигации в конструкторе бургеров (https://stellarburgers.education-services.ru/).

2. Описание проекта: Набор автотестов для веб‑приложения Stellar Burgers, проверяющий корректность пользовательского пути: от регистрации и входа в аккаунт до работы с конструктором бургеров.

3. Как установить проект:

   * **Шаг 1. Клонирование репозитория**  
     ```
     git clone https://github.com/ananna84-cmd/Sprint_5.git
     cd SPRINT_5
     ```
     *(переходит в папку проекта)*

   * **Шаг 2. Установка зависимостей**  
     ```bash
     pip install -r requirements.txt
     ```
     *(устанавливает все необходимые библиотеки)*

4. Как запустить проект:

   ```
   pytest -v tests/
   ```
  *(флаг -v обеспечивает подробный вывод результатов — статус каждого теста)*
